# AI Admin v2 - Финансовая модель

## Инструкция по созданию в Google Sheets

### Структура файла (6 листов):

1. **Dashboard** - Сводная панель с ключевыми метриками
2. **Inputs** - Входные данные (цены, тарифы, курсы)
3. **Unit Economics** - Юнит-экономика на 1 салон
4. **Scaling** - Сценарии масштабирования (1, 5, 15, 50, 100 салонов)
5. **LLM Comparison** - Сравнение LLM моделей
6. **Sensitivity** - Анализ чувствительности к изменениям параметров

## КАК СОЗДАТЬ В GOOGLE SHEETS:

### Шаг 1: Создание файла
1. Открой [Google Sheets](https://sheets.google.com)
2. Создай новую таблицу: "AI Admin - Финансовая модель"

### Шаг 2: Создание листов
1. Переименуй "Лист 1" → "Dashboard"
2. Создай новые листы: "Inputs", "Unit Economics", "Scaling", "LLM Comparison", "Sensitivity"

### Шаг 3: Заполнение данных
1. Начни с листа "Inputs" - скопируй все параметры
2. Затем "Unit Economics" - введи формулы
3. Затем "Scaling" - скопируй таблицу и формулы
4. Затем "LLM Comparison" и "Sensitivity"
5. В конце "Dashboard" - он будет тянуть данные из остальных листов

### Шаг 4: Форматирование
1. **Заголовки:** Жирный шрифт, фон серый
2. **Доходы:** Зеленый цвет текста
3. **Расходы:** Красный цвет текста
4. **Итоги:** Жирный шрифт, двойная граница сверху
5. **Проценты:** Формат "0.0%"
6. **Деньги:** Формат "# ##0₽" (с разделителем тысяч)

### Шаг 5: Графики (рекомендуемые)
1. **График 1:** Рост прибыли от количества салонов (линейный)
2. **График 2:** Структура расходов (круговая диаграмма)
3. **График 3:** Сравнение LLM моделей по цене (столбчатая)
4. **График 4:** Чувствительность прибыли к цене (линейный)

## ДОПОЛНИТЕЛЬНЫЕ ФИЧИ:

### 1. Условное форматирование:
- Прибыль > 100К → зеленый фон
- Прибыль 50К-100К → желтый фон
- Прибыль < 50К → красный фон

### 2. Выпадающие списки:
- Лист Inputs, B10: Выбор LLM модели (Data Validation)
- Лист Inputs, B35: Выбор УСН (1% или 5%)

### 3. Защита ячеек:
- Защити все ячейки с формулами
- Оставь редактируемыми только:
  - Inputs: B2 (количество салонов)
  - Inputs: B3 (цена)
  - Inputs: B6 (курс USD)
  - Inputs: B10 (выбор LLM)

## ПРИМЕЧАНИЯ:

1. **Курс USD/RUB:** Обновляй вручную или используй Google Finance функцию:
   ```
   =GOOGLEFINANCE("CURRENCY:USDRUB")
   ```

2. **Автоматический пересчет:** Все таблицы обновляются при изменении параметров в "Inputs"

3. **Сценарии "Что если":** Меняй параметры в "Inputs" и смотри изменения во всех листах

4. **Экспорт:** Можно экспортировать в Excel, PDF, CSV
