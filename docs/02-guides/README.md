## Guide Standards

All guides in this section follow these principles:

1. **Task-Oriented**: Focus on specific tasks and outcomes
2. **Step-by-Step**: Clear, numbered procedures
3. **Examples**: Real command examples and outputs
4. **Troubleshooting**: Common issues and solutions
5. **Quick Reference**: Summary cards for fast access
