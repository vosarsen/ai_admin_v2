ИНСТРУКЦИЯ ПО ПОДКЛЮЧЕНИЮ AI ADMIN

1. НАЖМИТЕ "ПОДКЛЮЧИТЬ" В MARKETPLACE
Вы будете перенаправлены на страницу подключения AI Admin

2. ПОДКЛЮЧИТЕ WHATSAPP САЛОНА (2 способа):

Вариант А - QR-код (рекомендуется):
• Откройте WhatsApp на телефоне салона
• Настройки → Связанные устройства → Привязать устройство
• Отсканируйте QR-код с экрана

Вариант Б - Код сопряжения:
• Введите номер WhatsApp → Получить код
• В WhatsApp: Настройки → Связанные устройства → Связать с номером телефона
• Введите полученный 8-значный код

3. АКТИВАЦИЯ (АВТОМАТИЧЕСКИ)
После подключения WhatsApp система автоматически:
✓ Отправит запрос активации в YClients API
✓ Синхронизирует данные салона (услуги, мастера, расписание)
✓ Отправит приветственное сообщение в WhatsApp

Время подключения: 20-30 секунд

ПРОВЕРКА: Отправьте в WhatsApp салона "Хочу записаться" - AI Admin ответит с предложением услуг
