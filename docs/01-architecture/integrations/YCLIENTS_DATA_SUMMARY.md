# AI Admin - Использование данных YClients (краткая справка)

## Где хранятся данные

**PostgreSQL** на выделенном сервере в **Санкт-Петербурге, Россия** (152-ФЗ)

## Какие данные получаем из YClients API

### 1. Компания (салон)
- Название, адрес, телефон, email
- Часовой пояс

### 2. Услуги
- Название, цена, длительность
- Категория услуги

### 3. Сотрудники (мастера)
- ФИО, специализация, должность
- Рейтинг

### 4. Расписание
- Свободные слоты для записи
- Время работы мастеров

### 5. Клиенты
- ФИО, телефон
- История визитов
- Скидки

### 6. Записи (бронирования)
- Дата, время, услуга
- Мастер, клиент
- Статус записи

## Для чего используются данные

AI Admin использует эти данные **только** для:
- Записи клиентов через WhatsApp
- Ответов на вопросы о услугах и ценах
- Напоминаний о визитах
- Персонализации общения (история предпочтений)

## Безопасность
